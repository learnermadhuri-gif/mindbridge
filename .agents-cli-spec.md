# MindBridge Agent Spec

## Overview

MindBridge is a **cognitive accessibility concierge** designed for neurodivergent users (ADHD, autism, anxiety, sensory processing differences). Unlike generic productivity assistants, it treats cognitive accessibility as a first-class design constraint: it adapts to the user's profile, breaks down overwhelming tasks, plans sensory-friendly outings, coaches social communication, and tracks energy/spoons.

## Track

**Concierge Agents** — a safe, useful personal assistant that simplifies everyday life while keeping user data secure and respecting the user's agency.

## Why It Is Unique

- Neurodivergent-first design rather than an afterthought.
- Profile-aware adaptation across task planning, sensory planning, communication, and energy accounting.
- Multi-agent architecture with specialized sub-agents for each domain.
- Privacy-first: all cognitive profiles and session data are stored locally (in-memory) by default.
- Safety guardrails: never diagnoses, always encourages professional support, avoids medical advice.

## Example Use Cases

1. **Task overwhelm** — User says "I have to clean my apartment and I can't start." The agent breaks the task into tiny steps, offers a timer, and suggests rewards.
2. **Sensory-friendly outing** — User asks "I need to run errands but I'm sensitive to noise and crowds." The agent plans a low-stimulation route and time.
3. **Communication coaching** — User says "I need to email my boss that I'm overwhelmed." The agent drafts an assertive but professional message and offers tone alternatives.
4. **Energy accounting** — User reports "I only have 3 spoons left today." The agent helps prioritize tasks and suggests rest.

## Architecture & Sub-Agents

- **root_agent** — Coordinates requests and routes to the right sub-agent.
- **task_breakdown_agent** — Decomposes overwhelming tasks into micro-steps with optional timers and rewards.
- **sensory_planner_agent** — Suggests low-stimulation environments, times, and coping strategies.
- **communication_coach_agent** — Drafts emails/messages with tone options for neurodivergent users.
- **energy_accountant_agent** — Tracks "spoons" and helps prioritize tasks.

## Model

- `gemini-2.5-flash-lite` — fast, cost-effective, and avoids the exhausted `gemini-3.5-flash` free tier.

## Tools Required

- `break_down_task` — Decompose a task into numbered micro-steps.
- `suggest_sensory_friendly_plan` — Recommend low-stimulation environment and timing.
- `draft_message` — Generate a message with requested tone.
- `track_energy` — Log energy/spoons and suggest priorities.
- `get_current_time` — Time-aware scheduling.

All tools are deterministic, local, and do not require external APIs.

## Constraints & Safety Rules

- Do not diagnose or provide medical advice.
- Include disclaimers for mental health or crisis situations.
- Respect user autonomy: offer options, not commands.
- Keep language non-judgmental and validating.
- Default to privacy: no cloud storage of personal profiles.
- Avoid urgency pressure; offer gentle pacing.

## Success Criteria

- Agent correctly identifies the user's domain (task, sensory, communication, energy).
- Responses are validating, non-overwhelming, and include concrete micro-steps.
- Safety guardrails trigger on mental health or crisis language.
- Agent adapts to stated cognitive profile (ADHD, autism, anxiety, sensory sensitivity).
- No cloud calls or external APIs required for core functionality.
